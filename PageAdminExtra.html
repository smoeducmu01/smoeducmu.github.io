🔗 ตั้งค่า Faculty API Room Integration
⏳ กำลังโหลดสถานะ...
📌 วิธีการทำงาน: เมื่อผู้ใช้เปิดหน้าจองห้อง ระบบจะเรียก API ของคณะเพื่อดึงข้อมูลการจองที่มีอยู่แล้ว แล้วนำมาตรวจสอบซ้อนกับข้อมูลในระบบชมรม เพื่อป้องกันการจองทับซ้อน

⚙️ การตั้งค่า API

ระบุ URL หลักของ API โดยไม่ต้องใส่ path เช่น /room-bookings
ชื่อห้องในระบบชมรม → room_id ในระบบคณะ (ขอจากทางคณะ)
🧪 ผลการทดสอบ
📋 ขั้นตอนการเชื่อมต่อ
1. ส่งเอกสาร API Spec ให้ทางคณะ (ดาวน์โหลดได้จากระบบ)
2. รอ API URL + API Key จากทางคณะ
3. ขอ Room ID Mapping (ว่าห้องไหนใช้รหัสอะไรในระบบคณะ)
4. กรอกข้อมูลในฟอร์มนี้แล้วกด "ทดสอบการเชื่อมต่อ"
5. เมื่อทดสอบผ่านให้ติ๊ก "เปิดใช้งาน" แล้วกด "บันทึก"
📧 ตั้งค่าอีเมลหน่วยงาน
📌 วิธีใช้: กำหนดอีเมลสำหรับแต่ละหน่วยงานและ Admin
เมื่อมีการยืมอุปกรณ์ ระบบจะส่งอีเมลแจ้งไปยังหน่วยงานที่มีอุปกรณ์ถูกยืม และ Admin โดยอัตโนมัติ
หมายเหตุ: หากต้องการเพิ่มหน่วยงานใหม่ ให้เพิ่มแถวตรงใน Google Sheet ชื่อ Email_Config

🚚 ผู้จ่ายของ / นัดรับอุปกรณ์

📋 ตรวจสอบข้อมูลก่อนยืนยัน

📜 เงื่อนไขการจองห้อง

ข้อ 1. ผู้จองมีหน้าที่ใช้ห้องตามวัตถุประสงค์ที่แจ้งไว้เท่านั้น ไม่นำไปใช้เพื่อวัตถุประสงค์อื่นโดยมิได้รับอนุญาต

ข้อ 2. ผู้จองต้องรักษาความสะอาดและความเรียบร้อยของห้อง จัดเก็บอุปกรณ์และของใช้ให้เป็นระเบียบก่อนออกจากห้อง

ข้อ 3. หากเกิดความเสียหายต่อทรัพย์สินภายในห้องอันเนื่องมาจากการใช้งาน ผู้จองรับผิดชอบชดใช้ค่าเสียหายตามมูลค่าจริง

ข้อ 4. ผู้จองต้องเข้าใช้ห้องตรงตามวันและเวลาที่จอง หากไม่สามารถใช้ได้กรุณาแจ้งยกเลิกล่วงหน้าอย่างน้อย 1 วันทำการ

ข้อ 5. ไม่อนุญาตให้โอนสิทธิ์การจองให้บุคคลอื่น หากพบจะถูกยกเลิกการจองและพิจารณาระงับสิทธิ์

ข้อ 6. ทีมงานขอสงวนสิทธิ์ในการยกเลิกหรือเปลี่ยนแปลงการจองในกรณีที่มีเหตุจำเป็น โดยจะแจ้งให้ทราบล่วงหน้า

* โปรดเลื่อนอ่านข้อความทั้งหมดจนสุด เพื่อให้ปุ่มยอมรับเงื่อนไขสามารถใช้งานได้
