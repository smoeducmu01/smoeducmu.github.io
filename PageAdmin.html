💬 LINE Users ผู้ลงทะเบียน LINE
📌 ขั้นตอนเชื่อมต่อ LINE Messaging API (ทำครั้งเดียว)
1. เข้า developers.line.biz → สร้าง Messaging API Channel → เลือก LINE OA ที่มีอยู่
2. คัดลอก Channel Access Token → ไปวางแทน LINE_TOKEN ใน Code.gs บรรทัดที่ 12
3. Deploy Web App → คัดลอก URL → วางใน Webhook URL ของ LINE Developers
4. เปิด Use webhook / ปิด Auto-reply / ปิด Greeting message
5. ผู้ใช้ Add เพื่อน LINE OA แล้วส่งรหัสนักศึกษา → ลงทะเบียนอัตโนมัติ
🔗 ตั้งค่า URL เว็บระบบ (สำหรับแสดงใน LINE)
URL นี้จะแสดงใน LINE เมื่อผู้ใช้กดเมนู "ยืมของ" หรือ "จองห้อง"
